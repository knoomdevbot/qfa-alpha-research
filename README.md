# qfa Alpha Research

Durable alpha research artifacts for the qfa/quant-for-agent workflow.

This repository holds research-only artifacts:

- `alpha_research/` — evolutionary research queue and issue notes.
- `models/` — alpha model source, config, metadata, and compact evaluations.

The qfa development repository is separate and should contain qfa source/tests/docs only.

Do not commit secrets, raw market data dumps, SQLite databases, caches, or paid/vendor datasets.
